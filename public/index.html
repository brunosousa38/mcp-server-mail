Serveur MCP Mail (IMAP/SMTP)

Ce site héberge un serveur MCP (Model Context Protocol) qui expose une boîte mail via IMAP et SMTP : lister les dossiers, lire, rechercher, envoyer et marquer des emails. Il est conçu pour être connecté à Claude (claude.ai, Claude Desktop) ou à tout client MCP compatible HTTP.

L'endpoint MCP est :

POST /mcp

L'accès est protégé par un token, à fournir soit dans l'en-tête Authorization: Bearer <token>, soit en paramètre d'URL ?token=<token> (utile pour les connecteurs personnalisés qui ne permettent pas d'en-têtes personnalisés).

Pour la configuration (serveurs IMAP/SMTP, déploiement Netlify) et la liste détaillée des outils disponibles, consultez le fichier README.md du dépôt.
